## 디스코드 음악봇 "Raccoon"

![다운로드](https://github.com/user-attachments/assets/d0f28f10-28d5-4baf-8a12-88a458aaf1a5)

### 제작 목적
- 최근 퍼블릭 음악봇들이 자주 고장나는 바람에 사람들이 불편을 호소함.
- 이로 인해 서버 고유의 음악봇을 의뢰 받아 제작하게 됨.

### 지원되는 기능
- 입장 : join
- 퇴장 : exit
- 재생 : play
- 볼륨 조절 : volume(0 ~ 100%)
- 음악 재생 중지 및 연결 해제 : stop
- 일시 중지 : pause
- 다시 재생 : resume
